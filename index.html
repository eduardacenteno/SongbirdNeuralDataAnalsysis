Songbird Data Analysis
A software that allows you to perform analysis on songbirds' neural and vocalization data stored as Spike2 format (.smr).
Software Documentation
This software was built with the aim of allowing the user to work with Spike2 .smr files and analysing neural and vocalization data collected from songbirds.
Our group works with neuronal and vocal data collected from Zebra finches while they sing. Therefore, the parameters here might have to be ajusted to your own specific data.

Requires:

* python 3.x.

* neo 0.7.1

* matplotlib 3.0.2

* numpy 1.15.4

* scipy 1.1.0

* pandas 0.23.4

* statsmodels 0.9.0

The code can be found in the following repository: https://github.com/aleblois/SongbirdNeuralDataAnalsysis
